Chrome 外掛使用

Chrome 外掛的使用可以讓你在瀏覽網頁的過程更快速有效地蒐集並記錄資料， 點擊 安裝連結 ， 登入以後會看到以下的介面 (目前只有提供 Google 登入)

狀態列

關聯 (左側)

顯示新增筆記時，隸屬的分類或是關聯的父筆記

使用者基本資料 (右側)

Mindnote 只會向 Google 拿到註冊使用者的 email、頭貼、稱呼

分頁

分類

分類分頁，提供你快速新增、刪除、設定瀏覽權限的功能，並且與網站資料同步。當你點擊其中一個分類時，可以讓未來新增的筆記隸屬於這個分類。

筆記歷史紀錄

透過 Chrome 外掛新增的筆記記錄

基本操作

在網頁選取文字以後按下
Command + Shift + P (Windows 按下 Ctrl + Shift + P) 會呼叫新增筆記的表單
Command + Shift + O (Windows 按下 Ctrl + Shift + O) 會把選取的文字塞到剪貼簿
